# 📦 Modules

Modules are reusable collections of Terraform resources grouped together to perform a specific function.

Instead of rewriting the same infrastructure repeatedly, modules allow engineers to define reusable building blocks that can be shared across multiple projects.

## Examples of common modules include:

- VPC
- EC2
- Security Groups
- IAM
- S3 Buckets
- RDS Databases
- Load Balancers

## Example:

```terraform
module "network" {
  source = "./modules/vpc"

  cidr_block = "10.0.0.0/16"
}
```

## Benefits of modules:

- Reusability
- Maintainability
- Standardization
- Scalability
- Easier collaboration

Large organizations often organize their Terraform code into reusable modules to reduce duplication and improve consistency across environments.
